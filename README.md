# Bankist App 

A modern banking application built using **HTML**, **CSS**, and **Vanilla JavaScript** as part of Jonas Schmedtmann’s Complete JavaScript Course.

This project simulates a real-world banking interface where users can log in, transfer money, request loans, close accounts, and view transaction history with formatted dates and currencies.

The project focuses heavily on advanced JavaScript concepts such as array methods, date handling, timers, internationalization APIs, and dynamic UI rendering.

---

## 🚀 Features

- User authentication/login system
- Display transaction history
- Money transfer functionality
- Loan request simulation
- Close/delete account feature
- Automatic logout timer
- Dynamic balance calculation
- Transaction summaries
- Currency formatting using Intl API
- Date and time formatting
- Sorting transactions
- Responsive banking dashboard UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
