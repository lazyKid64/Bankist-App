## 🌟 Overview

Bankist is a front-end banking application designed to demonstrate how modern JavaScript can be used to build dynamic, data-driven user interfaces without any frameworks.

Users can:

✅ Log into their accounts
✅ Transfer money
✅ Request loans
✅ Close accounts
✅ Track transaction history
✅ View balances and summaries
✅ Experience automatic session logout

All updates happen instantly through JavaScript-driven DOM manipulation.

---

## 🚀 Features

### 🔐 Secure Login System

* User authentication with username and PIN.
* Personalized welcome message after login.

### 💸 Money Transfers

* Transfer funds between existing accounts.
* Real-time balance updates.

### 🏦 Loan Requests

* Simulate bank loan approval.
* Automatic processing delay for realistic behavior.

### ❌ Close Account

* Permanently remove an account.
* Requires account credentials for confirmation.

### 📊 Financial Dashboard

* Current account balance.
* Total deposits.
* Total withdrawals.
* Interest calculations.

### 📅 Smart Date Formatting

* Displays transaction dates dynamically.
* Shows:

  * Today
  * Yesterday
  * X Days Ago
  * Actual Date

### 🌍 Internationalization Support

* Currency formatting using the **Intl API**.
* Locale-specific number formatting.
* Date formatting based on user locale.

### ⏱️ Automatic Logout Timer

* Session expires after inactivity.
* Improves security simulation.

### 🔄 Transaction Sorting

* Sort movements in ascending order.
* Instantly re-render transaction history.
